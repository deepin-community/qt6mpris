Qt and QML MPRIS interface and adaptor
======================================

[MPRIS v.2](http://specifications.freedesktop.org/mpris-spec/latest/index.html) specification implementation for Qt and QML plugin.


Installation:
-------------

```
$ qmake && make && make install
```
